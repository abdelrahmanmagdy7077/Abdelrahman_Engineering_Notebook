In this first project, the goal was to build a basic set of logic gates from scratch using only NAND gates. I went through and implemented the standard gates like NOT, AND, OR, and XOR, and then moved on to the 16-bit versions along with some data-routing components like multiplexers and demultiplexers. Overall, the project itself is really straightforward and easy to get through, but it serves as the necessary starting point for everything else that follows in the course.



Even though the logic itself wasn't difficult, the main takeaway for me was just getting comfortable with Hardware Description Language and seeing how it differs from writing normal software. In software, code executes sequentially, but in HDL, you are defining physical wiring and connections between gates that operate simultaneously. Aside from getting used to the syntax, this project was mostly a practical exercise in hardware abstraction. It showed how you can take one primitive component, build a few simple gates, and then immediately trust those gates to build more complex components like 16-bit multiplexers without constantly thinking about the underlying NAND connections.
