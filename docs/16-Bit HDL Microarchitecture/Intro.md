The purpose of this project is to understand how computers actually work from the ground up instead of treating them like a black box. I will be building an entire computer system step by step, starting with basic logic gates and working all the way up to a CPU, memory, an assembler, a compiler, an operating system, and a programming language.



This project is not just about building a computer. It is about understanding what each layer does and how everything fits together. I decided to pursue this project because one of my long term goals is to build a computer from scratch. When I started planning for that, I realized that I was missing a lot of the fundamental knowledge, especially when it came to computer architecture. Rather than skipping over those fundamentals, I decided to start with this project and build a solid foundation first.



For this project I will be working through the book \*The Elements of Computing Systems\*. My goal is not just to complete the project, but to gain a solid understanding of how computers work at every level so that I can confidently take on more advanced projects in the future, inshallah.
